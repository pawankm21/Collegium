# Collegium

An Web Application to manage college events.

  
## Run Locally

Clone the project

```bash
  git clone https://
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

  
## Tech Stack

**Client:** React, Tailwindcss, React-Router

**Server:** Node, Express, Bcrypt, Cloudinary SDK

**Database:** MongoDB Atlass
